Why was it built?

This is a project for the course Client Design & Development (ISTE-754) at RIT. The requirement was creating a client app using a JavaScript framework, that requested weather forecast data from a third-party API and then displayed the forecast for the next five days. The user must be able to specify the city for which the forecast should be displayed.
